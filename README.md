# Weekly Plan Mobile - Color Categories

Mobile-first weekly schedule and gym plan.

No photos. No GIFs.

Color categories:
- Yellow: Morning / Warm Up
- Blue: Work / Machine
- Green: Training / Dumbbell
- Purple: Reading / Core
- Orange: Dinner / Arms
- Gray: Rest / Sleep
